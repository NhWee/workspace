Blender Mantaflow FLIP wave tank
================================

Purpose
-------
This is the main high-realism path for the project. It replaces the browser
surface approximation with Blender's Mantaflow liquid solver, which uses a
FLIP-style liquid domain and can bake liquid mesh plus foam, bubble, and spray
secondary particles.

Scene
-----
The generated scene contains:

1. a 12 x 5 x 5 liquid domain
2. an initial liquid volume
3. an animated paddle that injects wave energy
4. a collision rock that creates breaking flow and splash
5. liquid mesh and foam/bubble/spray settings

Quality
-------
preview: resolution 112 for iteration.
long_preview: resolution 96, 360 frames, 48 fps. This is for a longer and
faster-to-watch driven wave tank preview.
production: resolution 256, higher mesh scale, and more time substeps. This is
an offline bake and may take a long time depending on cache frames and GPU.

Create a preview scene
----------------------
From the project root:

  .\tools\blender-5.1.2\blender-5.1.2-windows-x64\blender.exe --background --python wave_sim\blender_fluid\create_mantaflow_wave_tank.py -- --quality preview

Create and bake a longer fast-playback preview
----------------------------------------------

  .\tools\blender-5.1.2\blender-5.1.2-windows-x64\blender.exe --background --python wave_sim\blender_fluid\create_mantaflow_wave_tank.py -- --quality long_preview --output outputs\mantaflow_wave_tank_long_preview.blend --cache-dir outputs\mantaflow_wave_tank_long_preview_cache --bake

Optional overrides:

  --frame-end 480
  --fps 60

Higher fps makes the same baked frames play faster in Blender. Higher
frame-end makes the simulation longer, but also increases bake time and cache
size.

Create and bake the production scene
------------------------------------

  .\tools\blender-5.1.2\blender-5.1.2-windows-x64\blender.exe --background --python wave_sim\blender_fluid\create_mantaflow_wave_tank.py -- --quality production --bake

The blend scene is written to outputs/mantaflow_wave_tank.blend and its cache
is written to outputs/mantaflow_wave_tank_cache/.
