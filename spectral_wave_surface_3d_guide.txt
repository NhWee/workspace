spectral_wave_surface_3d.py 가이드
==================================

이 파일은 GPU FFT를 이용해 3차원 수면 표면 `z = eta(x, y, t)`를 만드는 spectral wave
prototype입니다.

기존 shallow-water solver는 수심과 유속을 직접 시간 적분합니다. 반면 이 solver는 주파수 공간의
파동 성분을 만들고, 각 성분을 deep-water dispersion relation으로 시간 진화시킨 뒤 inverse FFT로
수면 높이장을 복원합니다.


1. 사용한 모델
--------------

핵심 변수:

- eta(x, y, t): 수면 높이
- k = (kx, ky): 2D wave number
- |k|: wave number magnitude
- omega(k): 각 wave component의 angular frequency

사용한 dispersion relation:

    omega(k) = sqrt(g * |k|)

이는 deep-water gravity wave의 단순 모델입니다. 얕은물 bathymetry solver처럼 해안/섬/마른 셀을
다루지는 않지만, 넓은 수면에서 방향성을 가진 파도 표면을 빠르게 만드는 데 적합합니다.


2. 계산 흐름
------------

1. `torch.fft.rfftfreq`, `torch.fft.fftfreq`로 frequency grid 생성
2. dominant wavelength 근처에 Gaussian spectrum 생성
3. wind direction에 가까운 wave number에 더 큰 weight 부여
4. random complex noise에 spectrum amplitude를 곱해 초기 spectrum 생성
5. 각 저장 frame에서 `exp(i * omega * t)` phase로 spectrum 진화
6. `torch.fft.irfft2`로 height field 복원
7. 옵션으로 deep-water linear wave 근사의 surface orbital velocity `u, v` 복원
8. frame list와 flat depth를 `.npz` dataset으로 저장
9. 기존 Plotly 3D surface viewer로 HTML 생성

velocity를 저장할 때는 다음 근사를 사용합니다.

    u_hat(k, t) = -i * g * kx / omega(k) * eta_hat(k, t)
    v_hat(k, t) = -i * g * ky / omega(k) * eta_hat(k, t)

이는 표면 근처 orbital velocity를 보기 위한 실험용 근사입니다.


3. 실행 방법
------------

기본 실행:

    .\.venv\Scripts\python.exe spectral_wave_surface_3d.py

작은 테스트 실행:

    .\.venv\Scripts\python.exe spectral_wave_surface_3d.py --size 96 --steps 90 --frame-every 15 --output outputs\spectral_test.npz --viewer-output outputs\spectral_test.html

더 긴 파장:

    .\.venv\Scripts\python.exe spectral_wave_surface_3d.py --peak-wavelength 1.8 --wave-amplitude 0.07

방향성이 강한 파도:

    .\.venv\Scripts\python.exe spectral_wave_surface_3d.py --wind-direction-degrees 40 --directional-spread 12

속도장까지 저장:

    .\.venv\Scripts\python.exe spectral_wave_surface_3d.py --store-velocity --output outputs\spectral_wave_velocity.npz --viewer-output outputs\spectral_wave_velocity.html

속도 색상 viewer 생성:

    .\.venv\Scripts\python.exe shallow_water_velocity_viewer.py --input-npz outputs\spectral_wave_velocity.npz


4. 출력 파일
------------

기본 출력:

- outputs\spectral_wave_dataset.npz
- outputs\spectral_wave_viewer.html

NPZ에는 기존 dataset workflow와 같은 `frames`, `depth`, `metadata`가 들어갑니다.
따라서 `compare_wave_datasets.py`나 기존 Plotly viewer 입력으로도 재사용할 수 있습니다.


5. 왜 중요한가
--------------

이 solver는 목표인 3D wave surface에 더 가까운 방향입니다.

장점:

- FFT 기반이라 큰 grid에서도 GPU로 빠르게 계산 가능
- 방향성 있는 파도 표면을 자연스럽게 생성
- 기존 dataset, viewer, comparison workflow와 호환
- 향후 Phillips/JONSWAP spectrum, choppy wave, normal map, foam mask로 확장 가능


6. 현재 한계
------------

- deep-water dispersion만 사용합니다.
- bathymetry, shoreline, wet/dry mask는 없습니다.
- `--store-velocity`를 켜면 approximate surface orbital velocity를 `u_frames`, `v_frames`로 저장합니다.
- wave spectrum은 간단한 Gaussian directional spectrum입니다.
- energy conservation이나 실제 해양 스펙트럼 검증은 아직 하지 않았습니다.

8. 한 줄 요약
--------------

`spectral_wave_surface_3d.py`는 GPU FFT로 방향성 있는 3D 파도 표면을 생성하고, 기존 dataset과
Plotly viewer workflow로 바로 이어지는 spectral wave prototype입니다.
